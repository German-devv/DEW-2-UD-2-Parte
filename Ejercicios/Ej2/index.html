Ejercicio 2

El programa generará un número aleatorio comprendido entre 1 y 100 e invitará al usuario a adivinarlo. Si el usuario introduce un número mayor que el generado se le presentará el mensaje INTÉNTELO CON UN NÚMERO MENOR, en caso contrario INTÉNTELO CON UN NÚMERO MAYOR. Cuando el número sea adivinado se le indicará al usuario el número de intentos. Se limitará a 6 dicho número de intentos.
